# docsStarter

这是一个快速的文档生成器模版

This is a quick document generator template.


# 快速开始

## 克隆 docs-starter

```bash
git clone https://github.com/nest-center/docs-starter.git your-project-name
```

## 启动

```bash
npm run start
```

## 打开浏览器并访问

```bash
http://localhost:3000
```
